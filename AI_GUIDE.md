### Method 2: Use JavaScript Templates

```javascript
import { ai } from 'asciitecture/ai-templates';

// Create a simple button
const button = ai.button("Click me");

// Create a form
const contactForm = ai.form("Contact Us", ["Name", "Email", "Message"]);

// Create a dashboard layout
const dashboard = ai.dashboard("Admin Panel", ["Overview", "Users", "Settings"]);

// Combine multiple components
const layout = ai.combine(
  ai.navbar("MyApp", ["Home", "About", "Contact"]),
  ai.spacer(2),
  ai.grid(3, 2),
  ai.spacer(1),
  ai.button("Get Started")
);
```

## Best Practices for AI

1. **Keep It Simple**: ASCII wireframes should focus on structure, not visual design
2. **Use Standard Characters**: Stick to box-drawing characters (â”Œâ”‚â””â”€â”â”˜â•”â•‘â•šâ•â•—â•)
3. **Maintain Alignment**: Keep elements aligned to make the design clear
4. **Size Appropriately**: Don't exceed 100 characters wide or 50 lines tall
5. **Label Everything**: Add text labels to make component purposes clear
